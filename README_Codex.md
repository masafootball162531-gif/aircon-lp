# Codex作業メモ

このフォルダは、添付されたサイトデータから復旧した静的サイトです。

## 主な編集対象

- `index.html`: トップページ本体です。
- `thanks.html`: お問い合わせ後の完了ページです。
- `css/styles.css`: 見た目の調整を行うCSSです。
- `js/script.js`: 時刻表示、CTA計測、FAQ、固定ボタン、お客様の声スライダー、フォーム確認表示の動きです。
- `assets/`: ページ内で使用している画像です。

## 次回Codexに依頼するときの例

- `air01_codex_recovered の電話番号を変更して`
- `air01_codex_recovered のファーストビュー画像を差し替えて`
- `air01_codex_recovered のフォーム項目を追加して`
- `air01_codex_recovered をアップロード用にzip化して`

## 注意点

- `DS_Store` はサイト表示に不要なため、この復旧フォルダには入れていません。
- LINEリンクは現在 `https://line.me/R/ti/p/@example` の仮リンクです。実際のLINE公式アカウントURLが分かれば差し替えてください。
- `favicon.ico` は添付ファイルをそのまま配置していますが、元データが空ファイルの場合はブラウザタブのアイコンは表示されません。
